# articles_summary
Applies pre-trained LLMs to perform text tasks



## 0. Precondtions

- Create a virtual environment
```Bash
python3 -m venv nlpenv
source nlpenv/bin/activate
```


- Install Pytorch
```Bash
pip install torch
```


- Install the transformer library
```Bash
pip install transformers

pip install pandas
pip install chardet
```
